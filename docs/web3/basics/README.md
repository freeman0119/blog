# Web3 基础概念

## 什么是加密货币？

加密货币是一种基于密码学原理的数字或虚拟货币。它具有以下特点：

- 去中心化：不受任何中央机构控制
- 安全性：使用密码学确保交易安全
- 匿名性：交易方身份相对隐私
- 全球性：可以跨境自由流通

常见的加密货币包括比特币(BTC)、以太坊(ETH)等。

## 区块链与 Web3 之间是什么关系？

区块链是 Web3 的底层技术基础，两者关系如下：

1. 区块链提供：

   - 去中心化的数据存储
   - 不可篡改的交易记录
   - 智能合约的执行环境

2. Web3 基于区块链：
   - 构建去中心化应用(DApp)
   - 实现用户数据主权
   - 创建代币经济系统

## 什么是加密货币空投?

空投(Airdrop)是项目方免费向特定用户分发代币的营销方式：

- 目的：扩大用户基础、提高代币流通
- 方式：
  - 持仓空投：向特定代币持有者空投
  - 互动空投：奖励与项目互动的用户
  - 任务空投：完成特定任务获得奖励

## 什么是节点?

节点是区块链网络中的参与者，主要类型包括：

1. 全节点

   - 存储完整的区块链数据
   - 验证所有交易和区块
   - 维护网络安全

2. 轻节点

   - 只存储区块头信息
   - 依赖全节点验证交易
   - 适用于移动设备

3. 矿工节点
   - 参与区块生产
   - 获得出块奖励
   - 维护网络共识

## 什么是智能合约?

智能合约是运行在区块链上的自动执行的程序代码：

```solidity
// 简单的智能合约示例
contract SimpleStorage {
    uint256 private value;

    function setValue(uint256 newValue) public {
        value = newValue;
    }

    function getValue() public view returns (uint256) {
        return value;
    }
}
```

特点：

- 自动执行
- 不可篡改
- 透明公开
- 去中心化

## 什么是区块链交易手续费?

交易手续费是处理区块链交易时需要支付的费用：

1. 以太坊的 Gas 费

   - Gas Price：每单位 Gas 的价格
   - Gas Limit：交易最大可用 Gas
   - 总费用 = Gas Price × 实际使用的 Gas

2. 作用
   - 防止垃圾交易
   - 奖励矿工/验证者
   - 调节网络负载

## 什么是代币经济学？它的重要性何在?

代币经济学是设计代币发行、分配和激励机制的学科：

1. 核心要素

   - 代币供应机制
   - 通胀/通缩模型
   - 激励机制设计
   - 价值捕获模型

2. 重要性
   - 确保项目可持续发展
   - 平衡各方利益
   - 创造网络效应

## 什么是加密货币白皮书?

白皮书是加密货币项目的技术文档，通常包含：

1. 核心内容

   - 项目愿景和目标
   - 技术架构说明
   - 代币经济模型
   - 路线图和团队介绍

2. 著名案例
   - 比特币白皮书：定义了区块链技术
   - 以太坊白皮书：提出智能合约平台

## 什么是代币销毁?

代币销毁是永久性减少代币流通供应量的过程：

```solidity
// 代币销毁示例
function burn(uint256 amount) public {
    require(balanceOf[msg.sender] >= amount);
    balanceOf[msg.sender] -= amount;
    totalSupply -= amount;
    emit Transfer(msg.sender, address(0), amount);
}
```

目的：

- 通缩效应
- 增加稀缺性
- 提升代币价值

## 什么是工作量证明(PoW)?

工作量证明是一种共识机制，要求参与者证明他们完成了特定量的计算工作：

1. 工作原理

   - 解决复杂数学问题
   - 寻找满足条件的随机数
   - 证明计算工作量

2. 特点
   - 高度安全
   - 能源消耗大
   - 去中心化程度高

## 什么是代币标准?

代币标准定义了代币合约必须实现的接口和功能：

1. ERC-20（同质化代币）

```solidity
interface IERC20 {
    function transfer(address to, uint256 amount) external returns (bool);
    function balanceOf(address account) external view returns (uint256);
    // 其他标准接口...
}
```

2. ERC-721（非同质化代币）

```solidity
interface IERC721 {
    function ownerOf(uint256 tokenId) external view returns (address);
    function transferFrom(address from, address to, uint256 tokenId) external;
    // 其他标准接口...
}
```

## 什么是许可型区块链与非许可型区块链?

两种不同类型的区块链网络：

1. 非许可型（公有链）

   - 任何人都可以参与
   - 完全去中心化
   - 例如：比特币、以太坊

2. 许可型（联盟链/私有链）
   - 需要授权才能参与
   - 部分中心化
   - 例如：Hyperledger Fabric

## 什么是加密货币钱包?

加密货币钱包是管理加密资产的工具：

1. 类型

   - 热钱包：在线连接（MetaMask）
   - 冷钱包：离线存储（Ledger）
   - 软件钱包：应用程序
   - 硬件钱包：物理设备

2. 功能
   - 密钥管理
   - 交易签名
   - 余额查看
   - DApp 交互

## 什么是 51%攻击?

51%攻击是指单个实体控制超过网络一半算力的情况：

1. 风险

   - 可以篡改交易历史
   - 进行双重支付
   - 阻止新交易确认

2. 防御措施
   - 提高确认数要求
   - 使用 PoS 等其他共识机制
   - 跨链验证

## 双花问题详解

双花是指同一笔加密货币被重复支付的问题：

1. 发生原因

   - 区块链分叉
   - 网络延迟
   - 恶意攻击

2. 解决方案
   - 等待足够确认数
   - 使用零确认保护
   - 实时交易监控

## 什么保障了区块链的安全性?

区块链的安全性由多个层面保障：

1. 技术层面

   - 密码学算法
   - 共识机制
   - 分布式存储

2. 经济层面

   - 代币激励
   - 攻击成本高
   - 利益绑定

3. 网络层面
   - 节点分布广泛
   - P2P 网络韧性
   - 实时同步机制
